10 LA CAJA CSS

Prueba a modificar la regla CSS que afecta a todos los p para mostrar cambios en las cajas 1 y 2

¿Por qué los cambios en p no afectan al resto de cajas?

CAJA CSS 1
Caja con margen y padding de tamaño medio (50px y 20px). Es el estilo base.

CAJA CSS 2
Igual que la caja anterior: margen de 50px y padding de 20px. Sirve de referencia.

CAJA CSS 3
Sin margen exterior y con mucho padding (60px). El contenido queda muy separado del borde.

CAJA CSS 4
Margen exterior muy grande (100px) y poco padding (5px). Se aleja de las demás cajas.

CAJA CSS 5
Márgenes y rellenos asimétricos: arriba 10px, derecha 50px, abajo 80px, izquierda 20px.

CAJA CSS 6
Margen pequeño (10px) y padding medio (30px). Equilibrio entre separación y relleno.
